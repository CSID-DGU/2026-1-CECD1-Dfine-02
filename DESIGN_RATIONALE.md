### 2.3 군집 중심점 재조정 효과

**문제 상황**

- 기존 중심점(논문 기준): [5.7, 5.2, 4.4, 4.2, 4.3] 스케일
- 실제 데이터 평균: [3.6, 4.6, 3.6, 3.2, 2.8] 스케일
- **스케일 갭으로 인한 유사도 저평가 문제**

**해결책: MinMax 기반 중심점 재조정**

```python
# 원본 중심점의 비율 보존 + 현실 데이터 범위로 매핑
target_min = data_means - 1.5 * data_stds
target_max = data_means + 1.5 * data_stds
rescaled = ((orig - orig_min) / (orig_max - orig_min)) * (target_max - target_min) + target_min
```

**재조정 효과**

```
재조정 전:
- 평균 Top-1 유사도: 0.873
- Type1 제거율 (0.95 기준): 28.3%

재조정 후:
- 평균 Top-1 유사도: 0.991
- Type1 제거율 (0.95 기준): 5.1%
→ 허위 제거(False Positive) 80% 감소
```

## 3. 정합 매트릭스 통과 규칙 설정 이유

### 설정값

```python
def judge(row):
    chi_bad = row['Chi_Residual'] < CHI_THRESHOLD  # -1.5
    emb_bad = row['Emb_Gap'] < GAP_THRESHOLD        # 0.012

    if chi_bad and emb_bad:
        return '❌ INVALID'
    elif chi_bad or emb_bad:
        return '🟡 REVIEW'
    else:
        return '🟢 VALID'
```

### 의사결정 근거

### 3.1 이중 검증 시스템의 필요성

**단일 지표의 한계**

**Case 1: 통계적 잔차만 사용 시 문제**

```
Arch 0 × 적극적 소비: Chi_Residual = +3.43 (통계적 정합)
하지만 Emb_Gap = 0.0051 (의미론적 비정합)

→ 실제: "청년 무직"이 "적극적 소비"에 과다 매칭
→ 원인: 샘플 수 편향 (청년 무직이 1,203건으로 과대표)
```

**Case 2: 임베딩 갭만 사용 시 문제**

```
Arch 1 × 사회적 소비: Emb_Gap = 0.0188 (의미론적 정합)
하지만 Chi_Residual = +5.97 (통계적 과다 출현)

→ 실제: 중년 가장이 사회적 소비에 지나치게 집중
→ 원인: 데이터 수집 편향 (가족 중심 페르소나 과잉 생성)
```

**→ 두 지표가 상호보완적으로 작동**
