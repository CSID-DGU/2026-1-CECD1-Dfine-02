# 레이블 정의

파이프라인이 부여하는 2종 레이블(Tier 2 archetype, Tier 3 소비태그)의 이론적 근거와 실제 클러스터 특성을 정리한다.

---

## Tier 2 — Archetype (Schwartz BHV)

### 이론 배경

Schwartz(2012) Basic Human Values(BHV) 이론은 10개 보편 가치를 순환 모형(circumplex)으로 배치한다.  
인접한 가치는 동기적으로 호환되고, 반대 극의 가치는 상충한다.

```
       자기초월(universalism·benevolence)
              ↑
보존 ←──────────────→ 변화 개방
(conformity·tradition·security)  (self-direction·stimulation)
              ↓
       자기증진(achievement·power)
```

이 파이프라인은 10개 가치를 **인접 쌍 5종**으로 묶어 archetype을 정의한다.

### 5종 Archetype 정의

| 번호 | 레이블 | Schwartz 인접 쌍 | 핵심 동기 |
|------|--------|-----------------|---------|
| — | 안전·규범 | Security + Conformity | 안정·질서·규칙 준수, 예측 가능한 환경 유지 |
| — | 자애·안전 | Benevolence + Security | 내집단 돌봄 + 안정적 생활 기반 유지 |
| — | 전통·자애 | Tradition + Benevolence | 문화·관습 계승 + 공동체·가족 유대 |
| — | 성취·자율 | Achievement + Self-direction | 독립적 목표 추구 + 역량 인정 |
| — | 자율·자극 | Self-direction + Stimulation | 자유로운 선택 + 새로운 경험 추구 |

> 번호(cluster_id)는 KMeans 실행마다 달라질 수 있음. `archetype_mapping_n{N}.json` 참조.

### 실제 클러스터 특성 (n=200,000)

anchor v2 기준 Hungarian 매핑 결과:

**안전·규범 (cluster1, n≈56,254)**  
전문직(철도 관제사, 항공 정비사, 조색사, 건축 설계 보조). 매뉴얼 집착, 무사고 기록 경신, 전문성 심화. 자연 경관 휴양 여행(강원도·제주). 이자카야 혼술, 인라인스케이트.

**자애·안전 (cluster2, n≈43,292)**  
은퇴·노년층(전업주부, 사별 노인, 은퇴 공학도). 손주 카카오톡 사진 확인, 텃밭 가꾸기, 트로트 따라 부르기, 동네 친구들과 화투. 자녀와 수목원·온천·사찰 나들이.

**전통·자애 (cluster0, n≈26,205)**  
현장 노동자(청소원, 판매원, 폐기물 처리, 호송 경비원). 정년 유지와 소박한 경제적 자립. 경주·부여·공주 역사 유적지 나홀로 또는 가족 드라이브. 단골 주점에서 소주·꼬막무침.

**성취·자율 (cluster3, n≈41,403)**  
분석가·대학원생·마케터·스마트시스템 전문직. 엑셀 피벗·깃허브 코드 완벽주의. 맛집 리뷰 엑셀 정리 후 식도락 탐방(성수동·한남동). 스포츠 전술 분석, 자격증 준비.

**자율·자극 (cluster4, n≈32,846)**  
육아맘(독립서점·소품샵 꿈), 요양보호사(사회복지사 준비), 지역 공동체 참여. 취향 있는 소박한 자율성. 수목원, 골목 산책, 수국·제라늄 가꾸기, 잔잔한 플레이리스트, 모바일 퍼즐.

## Tier 3 — 소비태그

### 이론 배경

소비심리학·라이프스타일 세분화 연구에서 도출한 5개 소비 패턴. `culinary_persona`(음식 소비)와 `hobbies_and_interests_list`(여가 소비)로 측정한다.

### 5종 소비태그 정의

| 레이블 | 핵심 특성 | 행동 시그니처 |
|--------|---------|-------------|
| 절약·계획 | 예산 관리·절제 소비 | 카테고리별 식비 예산, 식재료 직접 구매·요리, 외식 월 1–2회 절제 |
| 경험·관계 | 사람 중심·관계 소비 | 친구·가족과 맛집 탐방, 단골 가게 사장님과 안부, 동호회 활동 |
| 가족·실용 | 가족 식탁 우선·건강 고려 | 된장찌개·나물 반찬 직접 요리, 가족 외식은 단골 식당, 손주·자녀 음식 준비 |
| 충동·즉흥 | 배달·야식·기분 전환형 | 배달 앱으로 치킨·족발·마라탕 즉흥 주문, 고깃집 삼겹살·소주, 야식으로 피로 해소 |
| 가치·정성 | 플레이팅·품질 중시 | 동네 유명 빵집 소금빵·단팥빵, 정갈한 한정식, 음식 색감과 플레이팅 중시 |

### 실제 클러스터 특성 (n=200,000)

anchor v2 기준 Hungarian 매핑 결과:

| KMeans cluster | 레이블 | cosine | 특성 요약 |
|---------------|--------|--------|---------|
| cluster1 | 절약·계획 | +0.183 | 식비 예산 관리, 나물 반찬·한식 직접 요리, 단골 사우나·박물관 |
| cluster2 | 경험·관계 | +0.081 | 노포·맛집 탐방, 친구·가족과 식사, 동호회·풍경 사진 촬영 |
| cluster3 | 가족·실용 | +0.382 | 가족 한식 요리, 텃밭 채소, 임영웅 감상, 온천 가족 여행 |
| cluster0 | 충동·즉흥 | +0.355 | 고깃집 삼겹살·소주, 배달 치킨·족발·마라탕, 동네 노래방 |
| cluster4 | 가치·정성 | +0.411 | 소금빵·단팥빵, 한정식 플레이팅, 베란다 홈가드닝, 일상 블로그 |

경험·관계(+0.081)가 낮은 것은 anchor 텍스트 분리 한계보다 **해당 클러스터 자체의 이질성**이 높기 때문일 가능성이 있다.

## 레이블 간 관계

Tier 2 archetype은 **가치관·삶의 방향**을 반영하고, Tier 3 소비태그는 **음식·여가 소비 패턴**을 반영한다. 두 축은 독립적으로 도출되므로 교차 분석이 가능하다.

### 5×5 정합 매트릭스 해석 (n=200,000, noise_dist=0)

```
          ctag0    ctag1    ctag2    ctag3    ctag4
arch0      3490     6362     5400     6173     2367    ← 전통·자애: 소비 이질성 높음
arch1     14478    14587    11140     3938     5311    ← 안전·규범: 균등 분포, 절약·계획 약간 우세
arch2      2948     6351     6897    20264     3608    ← 자애·안전: 가족·실용 압도적 집중
arch3     13505    11890     5016     1147     5382    ← 성취·자율: 충동·즉흥 우세
arch4      5006    10745     4648     3933     5413    ← 자율·자극: 절약·계획 약간 우세
```

- `arch2(자애·안전)–ctag3(가족·실용)` 20,264명: 은퇴·노년층의 가족 식탁 중심 소비와 일치
- `arch3(성취·자율)–ctag0(충동·즉흥)` 13,505명: 성취 욕구 높은 집단의 외식·배달 스트레스 해소
- `arch0(전통·자애)`: 모든 소비태그에 균등 분포 → 현장 노동자 계층의 소비 다양성
