# 실험 로그

파이프라인 파라미터 및 설계 결정을 검증하기 위한 실험 기록.  
각 실험은 독립 번호(Exp-NNN)로 관리한다.

---

## Exp-001 · Tier 3 PCA 차원 sweep

**목적**: 소비 임베딩(2048-dim) → UMAP(2D) → KMeans 파이프라인에서 최적 PCA 차원 결정  
**스크립트**: `src/sweep_consumption_pca.py`  
**평가 지표**: Hungarian 할당 후 min_cos / mean_cos / min_margin, 클러스터 balance

### Coarse sweep — n=50,000

```
uv run src/sweep_consumption_pca.py --sample 50000 --pca 10 30 50 100 200 500
```

| PCA | min_cos | mean_cos | min_margin | balance | 비고 |
|----:|--------:|---------:|-----------:|--------:|------|
|  10 | −0.0020 |   0.2596 |     0.0480 |   0.400 | min_cos 음수 — 정보 손실 과다 |
|  30 |  0.0254 |   0.2665 |     0.0265 |   0.421 | min_margin 최저 — 레이블 모호 |
|  50 |  0.0812 |   0.2826 |     0.0800 |   0.370 | 구 기본값 |
| **100** | **0.0966** | **0.2873** | **0.0862** | 0.369 | **전 지표 최고** |
| 200 |  0.0817 |   0.2848 |     0.0748 |   0.424 | 100 대비 전 지표 하락 |
| 500 |  0.0846 |   0.2729 |     0.0718 |   0.467 | mean_cos 하락 — noise 유입 |

**곡선 형태**: 10→100 단조 증가, 100→500 완만 감소. 명확한 최적점 존재.

**잠정 판정**: PCA=100 채택 (min_cos +0.015, min_margin +0.006 vs 기존 50)

> 참고: min_margin 최고(0.0862)가 Tier 2(0.27+)보다 크게 낮음. PCA 조정 외에 anchor 텍스트 재합성 여지 있음.

### Fine sweep — n=50,000

```
uv run src/sweep_consumption_pca.py --sample 50000 --pca 70 80 90 100 110 120 130 150
```

| PCA | min_cos | mean_cos | min_margin | balance | 비고 |
|----:|--------:|---------:|-----------:|--------:|------|
|  70 |  0.0853 |   0.2861 |     0.0737 |   0.375 | |
|  80 |  0.0916 |   0.2841 |     0.0808 |   0.360 | |
|  90 |  0.0948 |   0.2828 |     0.0823 |   0.370 | |
| **100** | **0.0966** | **0.2873** | **0.0862** | 0.369 | **전 지표 최고** |
| 110 |  0.0909 |   0.2868 |     0.0678 |   0.375 | min_margin 급락 |
| 120 |  0.0833 |   0.2852 |     0.0624 |   0.383 | |
| 130 |  0.0804 |   0.2851 |     0.0614 |   0.385 | |
| 150 |  0.0871 |   0.2860 |     0.0704 |   0.397 | |

**곡선 특이사항**: min_margin이 100→110 구간에서 0.0184 급락. min_cos는 90–100 구간에서 peak 후 완만 감소.

**anchor_sim PCA=100 상세** (Hungarian 할당 기준):

| 클러스터 | 레이블 | cosine |
|---------|--------|--------|
| cluster0 | 절약·계획 | +0.189 |
| cluster1 | 경험·관계 | +0.097 ← min |
| cluster2 | 가치·정성 | +0.412 |
| cluster3 | 가족·실용 | +0.377 |
| cluster4 | 충동·즉흥 | +0.362 |

경험·관계(+0.097)가 min_cos의 원인. 나머지 4개 anchor는 0.18 이상으로 양호.

**판정**: PCA=100 확정. coarse·fine 양쪽에서 전 지표 최고, 100→110 min_margin 급락으로 상한도 명확.
